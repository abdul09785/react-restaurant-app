# 🍽️ Restaurant App

This is my first React.js project where I built a simple Restaurant App UI. The main goal of this project was to practice React fundamentals by creating reusable components and displaying restaurant information in a clean layout.

## 🚀 Features

- Header with logo, search bar, and navigation menu
- Restaurant cards with image and details
- Footer section
- Clean and simple user interface
- Built using reusable React components

## 🛠️ Built With

- React.js
- JavaScript (ES6)
- HTML5
- CSS3
- Parcel

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/abdul09785/your-repository-name.git
```

Go to the project folder:

```bash
cd  Restaurant-App-React
```

Install the dependencies:

```bash
npm install
```

Run the project:

```bash
npx parcel index.html
```

Open your browser and visit:

```
http://localhost:1234
```
